SQLTags Tutorial

Welcome to the SQLTags Basic Tutorial. The goal of this tutorial is to get a new developer up and running with SQLTags using a professional J2SE 1.4 IDE or Integrated Development Environment and a relational database in the Windows environment as quickly as possible.

    This tutorial covers:
  • installation and setup of the development tools,
  • installation and setup of a relational database,
  • installation of the SQLTags web application,
  • running the SQLTags generator, and
  • authoring JSP pages using a generated tablib.

Lesson 1

Installation of the development environment. We will use Java 1.4.2 and the NetBeans IDE version 3.5.1 (COBUNDLED from Sun) as our Java development platform with MySQL version 4.0 as our relational database engine. A nice side-effect of this first lesson is a working JSP development environment.
